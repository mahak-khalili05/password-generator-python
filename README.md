# 🔐 Password Generator

A simple command-line password generator built with Python.

## ✨ Features

- Generate random passwords
- User chooses password length
- Supports:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Minimum password length validation

## 🛠️ Built With

- Python 3

## ▶️ Example

```
How many characters do you want? 12

Your password is: A7@kP2#Lm9$Q
```
